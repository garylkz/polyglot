# Usage

List of commands to run the file in its corresponding "mode".
 Do note that most example commands below is written in Bash.

## Bash

Make the file executable

```
chmod +x FILE
```

Run the file

```
./FILE
```

## C

Compile the file

```
gcc -x FILE
```

Run the file

```
./a.out
```

## HTML

Open the file with browser

## PHP

Run the file

```
php FILE
```

## Python

Run the file

```
python FILE
```

## Javascript

Run the file

```
node FILE
```
