# Usage

List of commands to run the file in its corresponding "mode".
 Do note that most example commands below is written for Bash.

## Bash

Make the file executable

```
chmod +x FILE
```

Run the file

```
./FILE
```

## C

Compile the file

```
gcc -x FILE
```

Run the file

```
./a.out
```

## HTML

Open the file with a browser

## Python

Run the file

```
python FILE
```

## Javascript

Run the file

```
node FILE
```
